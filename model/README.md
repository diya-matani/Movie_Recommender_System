### 2. Streamlit Cloud Deployment
This project is optimized for [Streamlit Cloud](https://streamlit.io/cloud).

1.  Push your code to GitHub (ensure `.gitignore` excludes large `.pkl` files).
2.  Connect your GitHub repo to Streamlit Cloud.
3.  **Configuration**:
    -   **Main file path**: `model/app.py`
    -   **Python version**: 3.10+
4.  **Deploy**: The app will automatically build the models during the first boot (approx 1-2 mins).
